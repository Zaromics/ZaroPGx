# ZaroPGx FHIR R4 Sample Exports

## Overview

This directory contains sample FHIR R4-compliant export files that demonstrate the structure of pharmacogenomic (PGx) reports generated by ZaroPGx. These samples follow the **HL7 Genomics Reporting Implementation Guide** and are compatible with **HAPI FHIR R4** servers.

**Reference:** [HL7 Genomics Reporting IG — Pharmacogenomics](https://build.fhir.org/ig/HL7/genomics-reporting/pharmacogenomics.html)

## Files

| File | Description |
|------|-------------|
| `sample_fhir_r4_pgx_bundle.json` | Complete FHIR R4 Bundle containing a sample PGx report with 3 genes and 3 drug recommendations |

## Bundle Structure

The sample Bundle (`type: collection`) contains the following FHIR resources in order:

### 1. DiagnosticReport (Genomic Report)
- **Profile:** `genomic-report`
- Top-level report that references all observations, links to the genomic study, and lists recommended actions via extensions
- Includes `conclusion` text and `conclusionCode` for overall report status

### 2. Patient
- **Profile:** `Patient`
- Identified by `urn:zaropgx:patient-id` system
- Includes optional demographics (name, gender, birthDate) when provided at export time

### 3. Observation — Genotype (one per gene)
- **Profile:** `genotype`
- Uses gene-specific LOINC codes (e.g., `79714-2` for CYP2D6, `79713-4` for CYP2C19)
- Components include:
  - **Gene studied** (`48018-6`) — HGNC gene symbol
  - **Phenotype** (`79716-7`) — e.g., "Intermediate Metabolizer"
  - **Activity Score** (`activity-score`) — numeric score from PharmCAT
  - **Haplotype Name** (`84414-2`) — one per allele (allele1, allele2)
- `valueCodeableConcept` carries the diplotype from PharmVar

### 4. Observation — Therapeutic Implication (one per drug)
- **Profile:** `therapeutic-implication`
- Components include:
  - **Medication assessed** (`51963-7`) — RxNorm-coded drug name
  - **Conclusion string** — clinical recommendation text
- `derivedFrom` links back to the genotype observation(s) that informed the implication
- Optional `extension` for related artifacts (CPIC/DPWG guideline citations)

### 5. Task — Medication Recommendation (one per recommendation)
- **Profile:** `medication-recommendation`
- `status: requested`, `intent: proposal`
- `description` contains the actionable recommendation text
- `for` → Patient reference
- `reasonReference` → links to the corresponding Therapeutic Implication
- `input` items for medication-assessed and evidence-level (Strong, Moderate, Optional)

### 6. Procedure — Genomic Study
- **Profile:** `genomic-study`
- SNOMED CT coded as "Genetic analysis" (`405824009`)
- LOINC coded as "Pharmacogenomic analysis panel" (`51969-4`)
- `note` records the PharmCAT version used

## HAPI FHIR Compatibility

This Bundle is designed for direct upload to a HAPI FHIR R4 server:

```bash
# Upload to HAPI FHIR
curl -X POST http://localhost:8080/fhir \
  -H "Content-Type: application/fhir+json" \
  -d @sample_fhir_r4_pgx_bundle.json
```

### Key compatibility notes:
- All resource IDs use valid UUID format
- All `fullUrl` entries use `urn:uuid:` scheme for Bundle-internal references
- Profiles reference the HL7 Genomics Reporting IG structure definitions
- The Bundle `type` is `collection` (standalone export, not a transaction)
- To use as a transaction Bundle for HAPI FHIR, change `type` to `"transaction"` and add `request` objects to each entry

### Converting to a Transaction Bundle

To POST as a transaction (creating all resources on the server):

```json
{
  "type": "transaction",
  "entry": [
    {
      "fullUrl": "urn:uuid:...",
      "resource": { ... },
      "request": {
        "method": "POST",
        "url": "DiagnosticReport"
      }
    }
  ]
}
```

## Code Systems Used

| System | URI | Usage |
|--------|-----|-------|
| LOINC | `http://loinc.org` | Observation codes, gene-specific codes |
| SNOMED CT | `http://snomed.info/sct` | Procedure category |
| PharmVar | `http://www.pharmvar.org` | Diplotype/haplotype codes |
| HGNC | `http://www.genenames.org/geneId` | Gene identifiers |
| RxNorm | `http://www.nlm.nih.gov/research/umls/rxnorm` | Medication codes |
| HL7 Genomics IG | `http://hl7.org/fhir/uv/genomics-reporting/CodeSystem/tbd-codes-cs` | IG-specific codes |
| ZaroPGx | `urn:zaropgx:*` | Internal identifiers |

## Generation

Sample exports are generated by `app/services/fhir_export_service.py` via:
- **API:** `GET /fhir/export/run/{run_id}` or `POST /fhir/export/run/{run_id}`
- **Report generator:** Automatically during report generation when `FHIR_EXPORT_ENABLED=true`

Real exports will contain all 23+ pharmacogenes analyzed by PharmCAT and all applicable drug recommendations from CPIC/DPWG guidelines.
